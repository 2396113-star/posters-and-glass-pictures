# Волшебные иллюстрации и витражи

Коллекция PNG-файлов для упаковки, витражей и печати.  
Каждая иллюстрация — это сказочный момент, созданный с любовью.

## Форматы
- 3000×3000 px
- 300 dpi
- PNG

## Темы
- Лилия у ручья
- Осенний лист каштана
- Ёжик с корзинкой ягод
